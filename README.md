# pete6pc.com

刘沛呈 · Peter Liu 的个人作品集网站，重点展示雅德士「小德」AI 内容工作流与 OpenClaw 中控台项目。

## 内容重点

- 首页：求职定位、核心成果、OpenClaw 中控台截图与简历下载。
- 作品集页：小德项目背景、业务约束、系统架构、生产迭代、踩坑记录和中控台升级。
- 简历：`Peter_Liu_Resume.pdf`，由 `tools/build_resume_pdf.py` 生成。

## 项目成果

- 内容生产耗时：近 8 小时人工工作压缩到 30 分钟选题与审核。
- 真实产出：23+ 篇雅德士公众号文章。
- 业务回报：带来公司近年首次央企主动询盘。
- 中控台能力：总览、内容库、Mode A/B 工作流控制、小德对话、OpenClaw 运维、发布中心、系统日志。

## 技术栈

- 静态 HTML / CSS / JavaScript
- React + TypeScript + Express 项目截图作为展示素材
- ReportLab 生成简历 PDF
- Vercel + GitHub 自动部署

## 部署

Push 到 `main` 分支后，Vercel 会自动更新 `pete6pc.com`。
